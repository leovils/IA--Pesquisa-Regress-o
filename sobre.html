Contexto pedagógico

O material foi desenhado originalmente para a disciplina Métodos e Técnicas de Pesquisa Quantitativa I (MTPQI) dos programas de pós-graduação da UNINOVE, e tem três objetivos pedagógicos principais:

  • Resolver a dificuldade comum dos alunos em distinguir significância estatística (p) de magnitude do efeito (R², f²), mostrando as duas grandezas lado a lado em todos os estágios.
  • Reduzir o tempo gasto pelos alunos com a escrita formatada de resultados (APA), oferecendo a frase pronta em cada análise para copiar e adaptar.
  • Permitir experimentação ao vivo com bancos próprios — o aluno sobe seu CSV e vê instantaneamente a regressão completa, os pressupostos e o relato APA.

Stack técnica

Material 100% standalone, sem dependências de servidor, banco de dados ou bibliotecas externas. Tecnologias usadas:

  • HTML5 + CSS3 + JavaScript puro (ES6+) — funciona em qualquer navegador moderno
  • Biblioteca estatística própria (stats.js) com cerca de 400 linhas — implementação JS pura de OLS, VIF, Breusch-Pagan, Durbin-Watson, Jarque-Bera, Cook's D, Pearson, distribuições t, F e χ². Validada contra statsmodels do Python — todos os números batem até a terceira casa decimal.
  • Animações SVG via CSS (@keyframes, transition) — sem GSAP, sem D3, sem jQuery
  • Hospedagem via GitHub Pages (gratuito, sem manutenção)
  • Analytics via Cloudflare Web Analytics (sem cookies, sem rastreamento individual, conforme LGPD por design)

Banco didático pré-carregado

O banco vem com 180 casos sintéticos e 6 variáveis: três preditores (VI1, VI2, VI3), uma moderadora (W1) e duas variáveis dependentes (VD1, VD2). Características relevantes para o ensino:

  • N = 180 — acima do mínimo de Green (1991) para até 9 preditores
  • Todos os pressupostos da regressão respeitados por construção (normalidade dos resíduos, homocedasticidade, independência, baixa multicolinearidade)
  • W1 amplifica os efeitos de VI1 sobre VD1 e VI2 sobre VD2 — útil para demonstrar moderação significativa
  • Seed reproduzível: 20260512

Os widgets também aceitam upload de CSV próprio — formatos brasileiro (separador ponto-e-vírgula, decimal vírgula) e internacional (separador vírgula, decimal ponto) são detectados automaticamente.

Quer adaptar para sua disciplina?

O material foi pensado para ser facilmente adaptável. Você pode trocar o banco didático (basta substituir assets/dados.js e ajustar os nomes das variáveis nos cinco HTMLs), modificar o estilo visual (paleta de cores está em variáveis CSS no assets/style.css), traduzir para outras línguas, ou estender com novos estágios. Em caso de dúvidas sobre customização, abra um issue no GitHub.
